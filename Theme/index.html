Tactico



Project Details

Tic-tac-toe is a classic game that unfortunately doesn't stay enjoyable for very long. After I had made the standard game work, I worked to implement a more strategic variation. With the Tactico variation, players take turns attacking and defending spaces rather than claiming them outright. Each player has to consider what the other player will do and play the "I know that she knows that I know..." game when trying to decide whether to go for the obvious play or not.

Potluck



Project Details

I hate deciding what I want for dinner. To remedy this problem, I built Potluck, a full-stack web application for storing recipes. Users can add as many ingredients and instructions as they like to a given recipe, search for recipes by name or ingredient, and even play "Recipe Roulette", a function that returns a random recipe from the database. Future versions will allow users more options for narrowing the possible recipes returned by necessary ingredients, excluded ingredients, and even by what's in their pantry.

Generic Assemblage



Project Details

Together with two other developers, I built this e-commerce site selling General Assembly themed t-shirts. My efforts were primarily spent getting the back end functioning properly, finding solutions for front end issues, and delegating tasks. Protection of user data has been an important factor in all projects shown here, but up until this point, the applications have required the user to log in to use the site. This didn't feel appropriate for an e-commerce site, so I implemented a "guest" user for each visit.
